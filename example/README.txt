This directory contains a toy example for running the entire Otedama pipeline. In order to run it, simply do the following:

cd $OTEDAMA_MAIN_DIRECTORY

./build.sh

cd example

./otedama.sh

The training and test data were taken from the Kyoto Corpus: https://alaginrc.nict.go.jp/WikiCorpus/index_E.html. Please note the copyright advisory: "NICT has created this corpus by translating Japanese Wikipedia articles into English and has made it publicly available under the conditions of Creative Commons Attribution-Share-Alike License 3.0. Users of the corpus are advised to read Wikipedia's copyright policy carefully to ensure proper usage."

The bilingual word alignments were created using Giza++: http://www.statmt.org/moses/giza/GIZA++.html
